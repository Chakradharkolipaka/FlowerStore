# FlowerStore

FlowerStore is a Vite + React flower shopping SPA with a Sepolia testnet payment flow using MetaMask.

## Repo layout

- `FlowerStores/` — frontend (Vite + React)
- `contracts/` — smart contracts (Solidity + Foundry)

## Deployment

Use the single guide: **`DEPLOYMENT_GUIDE.md`**.

## Contract documentation

See `contracts/README.md` for a short explanation of what the contract does, what testnet is used, what’s tested, and post-deploy checks.
